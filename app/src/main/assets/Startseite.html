Informationen

Diese App entstand für die Präsentation einer Bachelorarbeit.

Native Meldungen

Fehlermeldung anzeigen

Es wird eine native Fehlermeldung ausgegeben.

 Warnung anzeigen

Es wird eine native Warnung ausgegeben.

 Toast anzeigen

Es wird ein nativer Toast angezeigt.

 Push-Benachrichtigung senden

Die App sendet aus dem HTML-Code eine Push-Benachrichtigung an das Gerät.

Sensorfunktionen

Position anzeigen

Mittels GPS wird die Position ermittelt und diese auf einer Karte markiert.

 Foto machen

Eine Kamera wird geöffnet. Es kann ein Foto geschossen werden, das in die Gallerie gespeichert wird.

 Kurze Vibration

Der Vibrationsmotor wird angesetuert und eine kurze Vibration ausgeführt.

 Mittlere Vibration

Der Vibrationsmotor wird angesetuert und eine mittlere Vibration ausgeführt.

 Lange Vibration

Der Vibrationsmotor wird angesteuert und eine lange Vibration ausgeführt.

 Taschenlampe an

Wenn ein Kamera-Blitzlicht vorhanden ist, wird es eingeschalten.

 Taschenlampe aus

Wenn das Kamera-Blitzlicht an ist, wird es ausgeschalten.
